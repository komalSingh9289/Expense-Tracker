Welcome to the **Expense Tracker**!  
A smart and intuitive web application designed to help users manage expenses, understand spending habits, and improve financial health using **AI-powered insights**.

---

## ‚ú® Features

- **User Authentication**  
  Secure sign-up and login using JWT-based authentication.

- **Transaction Management**  
  Add, edit, and delete income and expense transactions easily.

- **Expense Categorization**  
  Organize transactions into meaningful categories for better analysis.

- **Search, Filter & Sort**  
  Quickly search transactions, filter by category, and sort by date.

- **Calendar-wise Transaction View**  
  View transactions organized date-wise for better monthly tracking.

- **AI Smart Suggestions & Financial Insights**  
  Get intelligent, easy-to-understand insights generated using AI based on your income and spending patterns.  
  The AI provides:
  - Savings percentage
  - Expense ratio
  - Spending behavior analysis
  - Practical suggestions on where to control expenses and how to manage savings better

- **Downloadable PDF Reports**  
  Export your transactions as a well-structured **PDF report**, useful for monthly or yearly review.

- **Responsive Design**  
  Fully optimized for desktop, tablet, and mobile devices.
